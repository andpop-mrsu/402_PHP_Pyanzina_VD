# Лабораторная работа 3. Игра «Наибольший общий делитель»

## Описание

Веб-приложение (Single Page Application) — игра «Наибольший общий делитель».

Игроку показываются два случайных целых числа. Нужно вычислить и ввести их наибольший общий делитель.
После ввода ответа выводится результат проверки и правильный ответ.

Все игры и ходы сохраняются в базу данных SQLite. Доступен просмотр истории всех игр с детализацией по ходам.

## Технологии

- **Backend:** PHP 8+, Slim 4, SQLite (PDO)
- **Frontend:** HTML, CSS, JavaScript (Fetch API)
- **Архитектура:** REST API + SPA

## REST API

| Метод | Путь         | Описание                                              |
|-------|-------------|-------------------------------------------------------|
| GET   | /games      | Получить список всех игр                              |
| GET   | /games/{id} | Получить данные игры с ходами по идентификатору        |
| POST  | /games      | Создать новую игру (передаётся имя игрока)            |
| POST  | /step/{id}  | Записать ход (два числа и ответ игрока)               |

## Требования

- PHP >= 8.0
- Расширение `ext-sqlite3`
- Composer
