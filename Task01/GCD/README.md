# Brain Games: GCD (Greatest Common Divisor)

[![Packagist Version](https://img.shields.io/packagist/v/iambadatnicknames/gcd)](https://packagist.org/packages/iambadatnicknames/gcd)

https://packagist.org/packages/iambadatnicknames/gcd

## Описание

Консольная игра «Наибольший общий делитель» (GCD).

Игроку показываются два случайных целых числа. Необходимо вычислить и ввести наибольший общий делитель этих чисел. Игра состоит из 3 раундов. Для победы нужно дать правильный ответ во всех раундах.

## Требования

- PHP >= 8.0
- Composer

## Установка

### Глобальная установка (рекомендуется)

composer global require iambadatnicknames/gcd
